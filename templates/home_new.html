🔌 API Endpoint Information

🎯 Main Endpoint
GET /like?uid=USER_ID&key=YOUR_API_KEY&server_name=SERVER
Parameters:
• uid - Player UID (required)
• key - Your API Key (required)
• server_name - Server (IND, BR, US, etc.) - optional
✨ Response Format
JSON Format with player information and like count
Includes:
• LikesGivenByAPI - Number of likes given
• PlayerNickname - Player's game name
• Region, Level, Status
🔐 Authentication
API Key based authentication required
How to get an API Key:
Contact the admin to request an API key for your application.

💡 Note: API requires a valid key to function. The key must be obtained from the admin.
